# KSO Download Turbo Ultra V1.0

برمجة: عبد الله و عبد الرحمن هاني [ KSO ]

برنامج تحميل ومذاكرة متكامل مبني بـ Python + PyQt6، مخصص للعمل على **Windows 10 Pro / 11**.
هذا مشروع سطح مكتب عادي — شغّله على جهازك مباشرة (لن يعمل داخل بيئة الويب هذه).

## المتطلبات قبل التشغيل

1. Python 3.10 أو أحدث (64-bit) من https://python.org
2. [FFmpeg](https://www.gyan.dev/ffmpeg/builds/) مضاف لمتغير النظام PATH (لازم لتحويل الصيغ وملفات MP3)
3. (اختياري) [aria2](https://github.com/aria2/aria2/releases) لو حابب تسريع أكبر عبر خيوط خارجية

## خطوات التشغيل

```bat
cd kso-download-turbo
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
python generate_assets.py
python main.py
```

## هيكل المشروع

```
kso-download-turbo/
├── main.py                 نقطة تشغيل البرنامج والواجهة الكاملة
├── generate_assets.py       ينشئ app_icon.ico (شغّله مرة واحدة قبل main.py)
├── lang.json                نصوص الواجهة بالعربي والانجليزي
├── config.json               إعدادات المستخدم (تُحفظ تلقائيًا)
├── requirements.txt          مكتبات بايثون المطلوبة
└── modules/
    ├── downloader.py         السرعة الذكية + محرك التحميل عبر yt-dlp
    └── compressor.py         تحويل الصيغ عبر FFmpeg
```

## المميزات

- متصفح مدمج (YouTube) مع اكتشاف تلقائي لروابط الفيديو والملفات
- بحث مباشر داخل يوتيوب مع "تحميل المزيد"
- سرعة تحميل ذكية تتكيف تلقائيًا حسب سرعة الإنترنت، أو تحديد يدوي
- مدير تحميلات باحترافية: تقدم %، سرعة، حجم، وقت متبقي، إيقاف/استكمال/حذف لكل صف
- جدولة التحميلات وتحديد سقف أقصى للسرعة (KB/s)
- محول صيغ مدمج عبر كليك يمين: MP3 / 720p / قص أول 30 ثانية (يتطلب FFmpeg)
- تبويب "المذاكرة": بحث عن كورسات، إنشاء مجلدات، تحميل بلايليست كاملة
- سجل تحميلات قابل للبحث وإعادة التحميل (`downloads_history.json`)
- وضع خفي (Ctrl+Shift+H) وحماية بكلمة سر اختيارية من الإعدادات
- دعم كامل للعربية والإنجليزية عبر زر تبديل اللغة (يتطلب إعادة تشغيل)
- إغلاق الجهاز تلقائيًا بعد انتهاء كل التحميلات (اختياري)

## اختصارات لوحة المفاتيح

| الاختصار | الوظيفة |
|---|---|
| Ctrl+Shift+K | إظهار البرنامج |
| Ctrl+Shift+Q | إغلاق البرنامج مع حفظ الحالة |
| Ctrl+Shift+D | تحميل الرابط الموجود في الحافظة |
| Ctrl+Shift+H | إخفاء البرنامج من شريط المهام (الوضع الخفي) |

## ملاحظات مهمة

- الاختصارات العامة (`keyboard`) على بعض إصدارات ويندوز تحتاج تشغيل البرنامج كـ Administrator لتعمل خارج نافذة البرنامج.
- تحويل الصيغ (MP3 / 720p / القص) يتطلب تثبيت FFmpeg وإضافته إلى PATH، وإلا ستظهر رسالة خطأ واضحة.
- زر "About" داخل البرنامج يعرض التعريف الكامل، الاختصارات، والمقارنة التفصيلية مع IDM و Snap Tube.
